# Editor E4 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add undo/redo (command pattern, 100 steps), complete keyboard shortcut mapping (Ctrl+1-5 panel focus, Ctrl+Shift+A add child), right-click context menus (scene tree + file browser), drag-drop (scene tree reparent, file→node lua_class), and full egui Visuals dark theme override with node type color icons.

**Architecture:** New `undo.rs` module. Modified `theme.rs` for visual overhaul. Modified `app.rs`, `scene_tree.rs`, `inspector.rs`, `file_browser.rs`, `behavior_editor.rs`, `agent_panel.rs`, and `state.rs` for undo integration, shortcuts, context menus, drag-drop, and icons.

**Tech Stack:** Rust 2024, egui 0.31, egui_dock 0.16. No new dependencies.

**Spec:** `docs/superpowers/specs/2026-07-18-editor-e4-design.md`.

### Undo wrap points:
- **Inspector** component value change: begin_action("change {key}") / add_do_undo / commit
- **SceneTree** add child (Ctrl+Shift+A): begin_action("add node") / add_do_undo / commit
- **SceneTree** delete: begin_action("delete node") / add_do_undo / commit
- **BehaviorEditor** JSON change: begin_action("edit behavior") / add_do_undo / commit
- **AgentPanel** accept diff: begin_action("apply agent suggestion") / add_do_undo / commit
